# Project 1: Intelligent News Summarizer & Sentiment Analysis System

This directory contains an Intelligent News Summarization and Sentiment Analysis system. It combines classical Natural Language Processing (NLP) tokenization and baseline sentiment scoring with state-of-the-art Large Language Model (LLM) reasoning via the Nararouter API and Mistral Medium 3.5.

---

## 🚀 Features
1. **📰 Flexible Input Options**: Input articles by scraping directly from a live web URL (utilizing `BeautifulSoup4`) or pasting the text manually.
2. **⚙️ Classical NLP Preprocessing**: Displays token counts and normalized text. It performs stopword filtering and tokenization using `NLTK`, displaying a beautiful interactive `Plotly` keyword-frequency barchart.
3. **📊 Abstractive Summarization**: Generates high-quality summaries using advanced LLM reasoning.
4. **⚖️ Hybrid Sentiment Analysis**: Compares traditional NLP sentiment scores (`NLTK VADER` baseline) side-by-side with semantic sentiment classification and breakdown generated by the LLM.
5. **💬 Grounded Q&A Chat**: Supports user query responses about the article, structured using ground-truth prompts to eliminate model hallucinations.

## ⚙️ Environment Setup & Installation (using `uv`)

This project is optimized to run inside an isolated Python virtual environment managed by `uv`.

### 1. Initialize the Environment & Install Dependencies
Run the following commands inside this directory to set up the Python environment:
```powershell
# Create an isolated virtual environment
uv venv

# Activate the virtual environment
# On Windows PowerShell:
.venv\Scripts\Activate.ps1

# Install requirements
uv pip install -r requirements.txt
```

### 2. Configure Credentials (`.env`)
Create a file named `.env` in this folder (one has already been automatically created for you by the agent) with the following parameters:
```env
NARAROUTER_API_KEY="your_actual_key_here"
NARAROUTER_BASE_URL="https://router.bynara.id/v1"
NARAROUTER_MODEL_NAME="mistral-medium-3-5"
```
*(If the `.env` parameters are ever missing, the application's sidebar will automatically offer secure text fields to input them manually).*

## 🏃 Running the Application

To launch the web interface, execute:

